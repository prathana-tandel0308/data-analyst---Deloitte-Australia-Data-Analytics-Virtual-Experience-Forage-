# Deloitte Australia Data Analytics Virtual Experience

## Project Overview
This project was completed as part of the Deloitte Australia Data Analytics Virtual Experience Program on Forage. The objective was to analyze operational and workforce data to generate actionable business insights using Tableau and Excel.

## Tasks Completed

### 1. Factory Telemetry Data Analysis
- Analyzed machine telemetry datasets to identify patterns in equipment performance.
- Investigated factory downtime and machine failure incidents.
- Created interactive Tableau dashboards for data visualization and decision-making.
- Generated insights to support operational efficiency improvements.

### 2. Pay Equality Analysis
- Performed workforce pay equality analysis using Microsoft Excel.
- Calculated and evaluated equality scores across employee groups.
- Categorized results into:
  - **Fair**
  - **Unfair**
  - **Highly Discriminative**
- Identified areas requiring attention to promote workplace equity.
